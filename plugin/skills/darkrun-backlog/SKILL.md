---
name: darkrun-backlog
description: Parking lot for ideas not yet ready to become a Run — add, list, review, or promote a backlog item into a Run
---

Call `darkrun_backlog` with an optional `action` (`list` (default) / `add` / `review` / `promote`) and `description`, then follow what it returns. When promoting an item into a Run, hand off to `/darkrun:darkrun-start` (or `/darkrun:darkrun-quick` for small items).
