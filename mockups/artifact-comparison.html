The single frontmatter shape, after review
  1. One rule everywhere: agent_type is the directory, never written. Every role file is just name + model + its slot-specific extras.
  2. Workers add an optional quality_gates (gates run in the pass-loop and at Audit).
  3. Explorers add a knowledge block — the shared-memory gap, closed.
  4. Stations drop position (path implies) and flat inputs (derived from the fixed flow); locked_artifact becomes a structured ref; elaboration and label are optional.
  5. Factories have no stations list — the six are a hardcoded mechanic; they add inherits and surfaces.
  6. Fix is a slot, not a type — a fix-worker is just a worker the station assigns. Reflections drop studio/agent_type. Templates become example runs.
  7. Still open: kills as a flat string vs a structured risk: block; and whether to keep the optional label at all (you asked what it's for — it's the domain display name over the fixed position).
